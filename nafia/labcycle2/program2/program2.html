Simple Selectors-Simple selectors target elements based on their name,id or class

Element Selector:Selects all element of a specific type

ID Selector(#my id):Selects a single element based on its id.An id is unique for each element

Class Selector(.my-class):Selects all element with a specific class

Combinator Selector:Combinator Selector selects element based on their relationship with other elements

Descent Selector(div p):It is used to select elements that are nested within other elements

Child Selector(div>p):Selects element that are children of specific element

Adjacent Sibling Selector(h1+p):Selects an element that is directly after a specified element

General Sibling Selector(h1~p):Selects all siblings of a specified element

Pseudo class selector:Psuedo class selector targets element in a specific state
